# TikSave

TikSave is a free macOS application for downloading TikTok media.

This repository is distributed as a binary-only release package.
No application source code is included.

## Contents
- `TikSave.dmg` — Compiled macOS installer
- `README.md` — Product and usage information
- `LICENSE` — License terms

## Installation
1. Download `TikSave.dmg` from the latest release.
2. Open the DMG file.
3. Drag `TikSave.app` into your Applications folder.
4. Launch TikSave from Applications.

If you need help or want to report an issue, open an issue in this repository.
