## 🚀 Installation

1. **Download** `TikSave-Installer.dmg` from this release
2. **Open** the DMG file
3. **Drag** TikSave.app to Applications folder
4. **Launch** TikSave from Applications or Launchpad

### First Launch

On first launch, macOS may show a security warning:
1. Right-click TikSave.app and select **Open**
2. Click **Open** in the dialog
3. Or go to **System Settings → Privacy & Security** and allow TikSave

---

## 📋 System Requirements

- **macOS**: 13.0 (Ventura) or later
- **Processor**: Apple Silicon (M1/M2/M3) or Intel
- **Memory**: 4 GB RAM minimum
- **Storage**: 50 MB for app + space for downloads
- **Internet**: Required for downloading videos

## 🎯 Quick Start Guide

### Download a Video

1. Copy a TikTok video URL
2. Paste into TikSave
3. Click the download button (arrow icon)
4. Video saved to your chosen folder!

### Supported URL Formats

```
https://www.tiktok.com/@username/video/1234567890
https://vm.tiktok.com/XXXXXXXXX/
https://vt.tiktok.com/XXXXXXXXX/
```

### View Profile Information

1. Go to **TikTok Stalker** tab
2. Enter a TikTok username
3. Click search (magnifying glass icon)
4. View detailed profile stats!

## 🐛 Known Issues

### Minor Issues
- DMG background image may not display on first mount (cosmetic only)
- Some TikTok URLs with special characters may need manual encoding

### Workarounds
- For DMG background: Eject and remount the DMG
- For special URLs: Copy the URL directly from TikTok app

## 🛠️ Troubleshooting

### App Won't Open
**Solution**: Right-click → Open, or allow in System Settings → Privacy & Security

### Download Fails
**Possible causes**:
- Internet connection issue
- Video is private or deleted
- TikTok API temporarily unavailable

**Solution**: Check connection, verify URL, try again

### Can't Change Output Folder
**Solution**: Ensure you have write permissions to the selected folder
