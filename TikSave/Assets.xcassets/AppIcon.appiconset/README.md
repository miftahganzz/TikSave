# TikSave App Icon Design Specification

## Design Concept
A minimalist, elegant app icon that represents TikTok video downloading with a "T" symbol and download arrow.

## Visual Requirements
- **Shape**: Rounded square (following macOS Big Sur+ design language)
- **Style**: Flat design with subtle gradient and soft shadows
- **Colors**: 
  - Primary: TikTok-inspired gradient (black to dark gray)
  - Accent: Bright cyan/blue for download elements
  - Background: Dark gradient for contrast

## Icon Specifications
- **16x16px**: Simplified version, minimal detail
- **32x32px**: Basic elements visible
- **128x128px**: Full design with details
- **256x256px**: High resolution version
- **512x512px**: Maximum detail for Retina displays
- **1024x1024px**: Source file for scaling

## Design Elements
1. **Background**: Rounded square with subtle gradient
2. **Main Symbol**: Stylized "T" representing TikTok
3. **Download Arrow**: Downward arrow integrated with "T"
4. **Optional**: Small video play triangle element

## Color Palette
- Dark Background: #1A1A1A to #2D2D2D gradient
- Primary Symbol: #FFFFFF
- Accent Elements: #00F2EA (TikTok cyan) or #25F4EE
- Shadow: Subtle black with 20% opacity

## Safe Area Guidelines
- Maintain 8px margin from edges for 512x512
- Critical elements within 448x448px center area
- Avoid placing important elements near corners

## Implementation Notes
- Use SF Symbols as reference for consistent styling
- Ensure readability at all sizes
- Test on both light and dark desktop backgrounds
- Follow Apple Human Interface Guidelines for app icons
